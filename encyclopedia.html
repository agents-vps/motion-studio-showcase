Micro-interaction • Tactile

1. Clic Physique Tactile 3D

CSS / GPU

Situation d'usage : Idéal pour les boutons d'appel à l'action principaux (validation de panier, soumission de formulaires critiques). L'effet de sillage 3D offre une confirmation physique instantanée en mimant l'élasticité d'un bouton mécanique réel.

Transition • Formulaires

2. Label Flottant Contextuel

CSS ONLY

Situation d'usage : Indispensable pour les formulaires d'inscription ou de connexion denses. Garder le label visible même lorsque le champ est rempli évite la surcharge cognitive et les erreurs de saisie de l'utilisateur.

Micro-interaction • Erreurs

4. Secousse d'Échec Physique

CSS TRIGGER

Situation d'usage : Indispensable pour signaler les mots de passe erronés ou les entrées incorrectes. Ce mouvement mimant le "non" humain attire instantanément le regard sur le champ bloquant sans nécessiter de long texte d'erreur agressif.

Révélations • Layouts

5. Accordéon CSS à Hauteur Variable

GRID LAYOUT

Situation d'usage : Idéal pour les sections FAQ denses ou les filtres de recherche complexes. La transition de hauteur s'adapte dynamiquement au volume de texte contenu sans nécessiter de calcul de hauteur en pixels via JS.

Un concept architectural où les structures physiques se déplacent et s'adaptent de manière réactive à l'environnement ou à l'activité des utilisateurs.

États • Perceived Latency

6. Shimmer Skeleton Loader

GPU PULSE

Situation d'usage : Utilisé lors du chargement initial de données asynchrones (cartes de produits, articles). Cela réduit la sensation d'attente de l'utilisateur par rapport à un indicateur de chargement rotatif classique.

Guidage • Attractivité

GSAP INERTIA

Situation d'usage : Idéal pour les boutons de navigation, les flèches directionnelles de sliders haut de gamme ou les boutons de lecture média. L'attraction subtile capte l'intérêt de l'utilisateur de façon spectaculaire.

Situation d'usage : Idéal pour configurer les interactions "bouncy" (comme l'ouverture de menus glissants ou le drag-and-drop). Gérer la raideur et l'amortissement donne un aspect organique et authentique.
