Native Top-Layer Popover

Top-Layer Promotion

This popover is rendered directly in the browser's Top Layer. It bypasses any parent `z-index` or overflow clipping entirely, animated smoothly with `@starting-style`.

No JavaScript libraries or coordinate math are required. Clicking outside dismisses the popover natively.

Exclusive Accordion Group

How is grouping achieved?

By placing the same `name` attribute on siblings. Opening one closes the other natively, animated cleanly via `::details-content`.

Is browser support complete?

Yes! This feature is fully supported in 2025/2026 as part of Baseline web standards.
